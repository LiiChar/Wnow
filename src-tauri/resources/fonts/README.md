# Шрифты для замены текста

## Требуемые шрифты

Для работы функции замены текста необходим шрифт в формате TTF.

### Рекомендуемый шрифт: Noto Sans

Noto Sans — это шрифт без засечек, поддерживающий множество языков, включая кириллицу.

**Скачать:** https://fonts.google.com/noto/specimen/Noto+Sans

### Установка

1. Скачайте шрифт Noto Sans Regular (TTF)
2. Поместите файл в эту директорию
3. Переименуйте файл в `NotoSans-Regular.ttf`

Или используйте любой другой шрифт TTF, обновив путь в коде.

## Альтернативные шрифты

- **Roboto** — https://fonts.google.com/specimen/Roboto
- **Open Sans** — https://fonts.google.com/specimen/Open+Sans
- **Arial** — системный шрифт Windows

## Примечание

Без шрифта функция замены текста не будет работать. Убедитесь, что файл шрифта присутствует в директории перед сборкой приложения.
