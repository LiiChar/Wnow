# 🌐 Wnow

**Wnow** — это десктопное приложение-переводчик с функциями изучения слов. Переводите текст с экрана, сохраняйте слова в словарь и изучайте их с помощью системы интервальных повторений.

![Версия](https://img.shields.io/badge/version-0.1.0-blue.svg)
![Tauri](https://img.shields.io/badge/Tauri-2.0-24C8DB?logo=tauri)
![Solid.js](https://img.shields.io/badge/Solid.js-1.9-2c4f7c?logo=solid)
![Rust](https://img.shields.io/badge/Rust-1.70+-orange?logo=rust)

---

## ✨ Возможности

- 🔍 **Перевод текста с экрана** — OCR + перевод любой области экрана
- 🎯 **Перевод слова под курсором** — по горячей клавише
- 📐 **Перевод выделенной области** — рисуйте полигоны для точного выделения
- 📋 **Перевод из буфера обмена** — выделите текст и нажмите Ctrl+Shift+C
- 📚 **Словарь** — автосохранение переведённых слов
- 🎓 **Изучение слов** — flashcards с алгоритмом интервального повторения (SM-2)
- 🌐 **Офлайн-перевод** — локальные ML-модели + онлайн-перевод (Google Translate)

## ⌨️ Горячие клавиши

| Действие | Комбинация |
|----------|------------|
| Перевод слова под курсором | `Ctrl+U` |
| Выделить область для перевода | `Ctrl+Y` |

## 🚀 Быстрый старт

### Требования

- **Node.js** >= 18
- **Rust** >= 1.70
- **cargo-tauri**: `npm install -g @tauri-apps/cli`

### Установка

```bash
# Клонируйте репозиторий
git clone https://github.com/your-username/wnow.git
cd wnow

# Установите зависимости
npm install
```

### Запуск в режиме разработки

```bash
# Запуск Vite dev-сервера
npm run start

# В новом терминале: запуск Tauri приложения
npm run tauri dev
```

### Сборка релиза

```bash
# Сборка фронтенда
npm run build

# Сборка нативного приложения
npm run tauri build
```

## 🏗️ Архитектура

Wnow построен на стеке **Tauri 2 + Solid.js + TypeScript + Rust**:

```
┌─────────────────────────────────────────┐
│         Frontend (Solid.js)             │
│  • Главное окно (словарь, настройки)    │
│  • Overlay окно (перевод поверх экрана) │
│  • Notification окно (уведомления)      │
└─────────────────┬───────────────────────┘
                  │ Tauri API
┌─────────────────▼───────────────────────┐
│          Backend (Rust)                 │
│  • OCR (ocr-rs)                         │
│  • Перевод (CTranslate2, Google)        │
│  • SQLite база данных                   │
│  • Глобальные горячие клавиши           │
└─────────────────────────────────────────┘
```

## 🎨 Технологии

### Frontend
- **Solid.js** ^1.9.3 — реактивный фреймворк
- **@kobalte/core** — UI компоненты
- **Tailwind CSS** — стилизация
- **@solidjs/router** — роутинг

### Backend
- **Tauri** v2 — фреймворк для десктопных приложений
- **ocr-rs** — OCR распознавание
- **CTranslate2** — быстрый перевод
- **rusqlite** — SQLite база данных
- **enigo, rdev** — управление вводом

---

## 📚 База данных

Wnow использует SQLite для хранения:

- **words** — слова, перевод, уровень владения, интервалы повторений
- **settings** — настройки приложения
- **stats** — статистика изучения
